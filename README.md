# Eximius-2k27

## Overview

A class website for the members of UI Eximius 2k27 class addressing their general needs and wants. The early-stage will allow for quizzes to be held on it.

## Tech Stack

### Backend

- Python
- Django

### Frontend

- HTML
- CSS

### Tools

- Pipenv

## Project Structure

config/
core/
quiz/
manage.py

## Setup Instructions

- git clone repo
- pip install pipenv
- pipenv shell
